# BunRepo
bunを学習・調査するためのリポジトリです。

## コマンド

- テンプレートプロジェクト作成

    ```bash
    bun create vite my-app
    ```

- インストール

    ```bash
    bun install
    ```

- ビルド

    ```bash
    bun run build
    ```

- 起動

    ```bash
    bun run dev
    ```
